# Linux Basics

## What is Linux?
Linux is an open-source operating system widely used in servers and cybersecurity.

## Why Linux is important for cybersecurity
- Most security tools run on Linux
- Kali Linux is based on Linux
- Better control over system and networking

## Common Linux commands
- ls : list files
- pwd : show current directory
- cd : change directory
- mkdir : create folder
- rm : delete files
